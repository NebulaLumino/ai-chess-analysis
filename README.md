# AI Chess Analyst

An AI-powered AI Chess Analyst tool built with Next.js 16, TypeScript, Tailwind CSS, and DeepSeek API.

## Features
- Generate AI Chess Analyst content instantly
- Dark gradient UI
- Powered by DeepSeek AI
- Fully responsive

## Setup
```bash
export OPENAI_API_KEY=your_deepseek_api_key
npm install && npm run dev
```
